# Trivago-Extension
----
Steps to clone repository:

1.- git clone https://github.com/jojo5716/Trivago-Extension

2.- go to chrome extension, activate developer mode and add this folder

3.- go to Trivago.es and make a search.

4.- Enjoy! =)
